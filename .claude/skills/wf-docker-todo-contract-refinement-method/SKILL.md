# Method: TODO Contract Refinement

Use after lane/framing confirms a tactical TODO is the governing artifact. Canonical details live in `workflows/docker/todo-contract-refinement-method.md`.

## Responsibilities
- Align status schema, scope, DoD, validation, blocker/provisional fields.
- Confirm module anchors, execution profile, technical scope, and handoff log.
- Triage gaps into material decisions, implementation details, or redundant items.
- Record complexity and checkpoint cadence.
- Build assumptions, execution plan, and required evidence/consumer/CI matrices.

## Outputs
- Refined TODO ready for approval gates.

## Non-Negotiables
- No approval request while material decisions or required matrices are missing.
- No guessed assumptions.
