# Gemini Bootloader

1. Read `./delphi-ai/main_instructions.md` before any work; this is the primary instruction source.
2. For downstream project work, run `bash delphi-ai/verify_context.sh` as a readiness check.
3. Skills are available in `.agents/skills/`.
4. Rules are in `.agents/rules/`.
5. Workflows are in `.agents/workflows/`.

## Compatibility

This repo also supports:
- **Claude Code**: See `CLAUDE.md`, `.claude/rules/`, `.claude/skills/`, and `.claude/settings.json`
- **Codex/Antigravity**: See `AGENTS.md` and `.codex/skills/`
- **Cline**: See `CLINE.md`, `.clinerules/`, and `.cline/skills/`
