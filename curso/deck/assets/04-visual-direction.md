# Visual Direction

## Intencao
O site precisa parecer um **artefato editorial premium**, nao um conjunto de slides genericos nem um dashboard SaaS.

## Direcao visual
- sofisticado
- intelectual
- limpo
- forte
- contemporaneo

## Sensacao desejada
- clareza com peso
- densidade com respiracao
- tecnologia com criterio

## O que evitar
- roxo neon
- vidro generico
- dark mode cyberpunk por default
- gradientes clichês de IA
- cara de template
- excesso de cards iguais
- visual "pitch deck startup"

## Tipografia
Evitar stack default tipo Inter/Roboto/Arial como base unica da identidade.

Direcao sugerida:
- serif ou semi-serif expressiva para manchetes
- grotesk/sans limpa para corpo e navegacao

Combinacoes aceitaveis:
- Fraunces + Space Grotesk
- Newsreader + Manrope
- Instrument Serif + IBM Plex Sans

## Cor
Paleta com base clara por default.

Direcao sugerida:
- fundo off-white / paper
- texto quase-preto
- 1 cor profunda principal
- 1 cor de apoio para dados ou destaque

Exemplos de sensacao:
- areia + tinta + verde escuro
- marfim + grafite + azul petroleo
- papel quente + vinho escuro + oliva

## Layout
- hero com composicao forte
- assimetria controlada
- grids editoriais
- espacamento generoso
- linhas, blocos, divisores e indices discretos

## Motion
- sutil
- transicoes curtas
- revelacao progressiva de conteudo
- nada de animacao gratuita

## Componentes desejados
- tela principal tipo full-screen section
- callouts tipograficos fortes
- bloco lateral para links de aprofundamento
- source badges discretos
- navegacao inferior ou lateral elegante

## Relacao entre deck e site
Na navegacao principal, a tela precisa se comportar como slide.
No aprofundamento, a interface pode abrir conceito/caso/fonte com mais texto e menos teatralidade.

## Regra de ouro
O visual precisa ajudar a vender a ideia de:
- metodo
- criterio
- engenharia
- maturidade

Nao pode parecer:
- marketing de ferramenta
- landing page de produto
- template genérico de IA
