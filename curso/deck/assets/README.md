# AI Studio Assets

Arquivos pensados para upload no Google AI Studio junto com o prompt principal.

## Ordem sugerida
1. `01-product-brief.md`
2. `02-information-architecture.md`
3. `03-aula-1-screen-map.md`
4. `04-visual-direction.md`
5. `05-core-concepts-and-sources.md`
6. `google-ai-studio-prompt.md`

## Uso
- Faça upload dos arquivos `01` a `05`.
- Cole o conteúdo de `google-ai-studio-prompt.md` no prompt principal do AI Studio.
- Se o AI Studio permitir anexar vários arquivos, envie todos os arquivos `01` a `05` juntos.
- Trate os arquivos anexados como fonte de verdade editorial.
