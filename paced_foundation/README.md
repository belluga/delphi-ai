# PACED Foundation

This directory is reserved for **PACED-internal documentation surfaces**.

It exists to separate framework-level artifacts from the downstream project's
`foundation_documentation/` package.

Use this namespace for:

- PACED-internal strategic artifacts
- PACED-internal roadmap or constitutional material, if/when promoted
- framework-level discussion documents that should not be mistaken for
  downstream project canon

Do not use this directory for:

- downstream project constitution
- downstream project roadmap
- downstream project modules
- downstream project tactical TODOs

Those remain under the downstream repository's own
`foundation_documentation/` package.
